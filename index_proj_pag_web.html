Tag H

(Voltar)

A tag H1 é um elemento HTML que define o título principal de uma página da web. É considerada a tag mais importante de uma página, pois é um fator de classificação para os motores de busca e é normalmente o conteúdo mais visível para o leitor. As tags H1, H2, H3, H4, H5 e H6 são chamadas de heading tags e são recursos de programação que definem os títulos e subtítulos de uma página. Cada uma dessas tags representa um nível de título de seção, sendo H1 o mais alto e H6 o mais baixo. As heading tags são importantes porque: Fornecem uma estrutura hierárquica ao conteúdo, facilitando a leitura e compreensão Ajudam os motores de busca a entender a estrutura e a relevância do conteúdo Melhoram a acessibilidade da web, permitindo que os usuários naveguem rapidamente pelos diferentes tópicos e seções.

Tag p

(Voltar)

O elemento HTML p representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco

Tag mark

(Voltar)

A tag HTML mark é usada para marcar ou destacar o texto de interesse ou relevância especial em um documento HTML. A tag mark é nova no HTML 5. Os navegadores tradicionalmente renderizam o texto encontrado na tag mark como texto com uma cor de fundo amarela.

Tag small

(Voltar)

A tag HTML small é usada para definir um texto menor, como comentários laterais ou direitos autorais. No entanto, no HTML5, a tag small também é usada para marcar restrições legais, avisos obrigatórios ou restrições de uso de conteúdo.

Tag i

(Voltar)

O elemento HTML i representa uma parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios. Normalmente, é apresentado com o uso do tipo "itálico".

Tag strong

(Voltar)

O elemento HTML strong dá ao texto uma forte importância, e é tipicamente mostrado em negrito.

Tag ol

(Voltar)

A tag HTML ol serve para criar uma lista ordenada de itens. Os itens são apresentados com uma contagem que os precede, podendo ser números, letras, algarismos romanos ou símbolos.

Tag ul

(Voltar)

A tag ul é um elemento HTML que representa uma lista não ordenada, ou seja, os itens são apresentados com marcadores.

Tag li

(Voltar)

A tag HTML li serve para representar um item de uma lista. Ela deve estar contida em um elemento pai, como uma lista ordenada (ol), uma lista desordenada (ul) ou um menu (menu). As tags ol e /ol delimitam uma lista ordenada, enquanto as tags li e /li delimitam cada item da lista.

Tag a

(Voltar)

A tag a (âncora) é usada na linguagem HTML para criar links que conectam uma página a outra ou a outros recursos na web. Por exemplo, uma âncora pode apontar para uma imagem, um arquivo de som, um filme ou outra página em HTML

Tag hr

(Voltar)

A tag hr do HTML serve para inserir uma quebra temática ou uma linha horizontal em um documento web

Tag sub

(Voltar)

A tag sub do HTML serve para formatar um trecho de texto como subscrito, ou seja, renderizar o texto em um alinhamento vertical mais baixo em relação ao texto principal.

Tag sup

(Voltar)

A tag HTML serve para formatar um trecho de texto como sobrescrito, ou seja, renderizar o texto em um alinhamento vertical mais alto em relação ao texto principal. É muito útil em fórmulas químicas e em algumas equações matemáticas

Tag blockquote

(Voltar)

A tag blockquote, serve para indicar que o texto incluído é uma longa citação no HTML.

Tag font

(Voltar)

A tag font é uma abreviação que permite declarar mais de uma propriedade para fonte no CSS. É possível declarar todas ou algumas das propriedades, mas os valores size e family são obrigatórios.

Tag del

(Voltar)

A tag del é um elemento HTML que indica que uma parte do texto de um documento foi excluída.

Tag abbr

(Voltar)

A tag HTML abbr (ou Elemento de Abreviação HTML) serve para representar uma abreviação e, opcionalmente, fornecer uma descrição completa para ela. É geralmente utilizada como item de um glossário ou em uma explicação didática.
